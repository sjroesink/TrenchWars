  <name>Weapon Physics (Bullets and Bombs)</name>
  <files>packages/shared/src/weapons.ts, packages/shared/src/__tests__/weapons.test.ts</files>
  <behavior>
    - createBullet: Given a ship at (10,10) facing east (orientation=0) with vx=0,vy=0 and bulletSpeed=12.5 tiles/s, bullet.vx should be 12.5, bullet.vy should be 0. bullet.x=10, bullet.y=10, type='bullet', bouncesRemaining=0.
    - createBullet with ship velocity: Ship at (10,10) facing east, vx=5, bulletSpeed=12.5 -> bullet.vx=17.5 (inherits ship velocity).
    - createBomb: Given ship facing east with bombBounceCount=3, bomb.bouncesRemaining=3, type='bomb'.
    - createBomb recoil: After creating bomb, the returned recoil value should be -bombThrust in heading direction.
    - updateProjectile bullet: Bullet at (10,10) with vx=12.5,vy=0 after dt=0.01 -> x=10.125, y=10. Returns 'active'.
    - updateProjectile bullet wall: Bullet hitting wall returns 'wall_explode' (bullets don't bounce).
    - updateProjectile bomb wall: Bomb hitting wall with bouncesRemaining=3 -> bouncesRemaining decrements to 2, velocity reverses on colliding axis, returns 'active'.
    - updateProjectile bomb wall final: Bomb hitting wall with bouncesRemaining=1 -> returns 'wall_explode'.
    - checkProjectileHit: Bullet at (10,10) with radius=0.1, player at (10.3,10) with radius=0.875 -> overlap -> hit=true.
    - checkProjectileHit miss: Bullet at (10,10), player at (15,15) -> no overlap -> hit=false.
    - checkProjectileHit self: Bullet ownerId='p1', player id='p1' -> always false (can't hit self).
    - calculateBombDamage: At center (distance=0), damage = BOMB_DAMAGE_LEVEL = 550. At edge (distance=BOMB_EXPLODE_RADIUS), damage = 0. At half radius, damage = 275.
  </behavior>
  <implementation>
Create `packages/shared/src/weapons.ts` with these pure functions:

1. `createBullet(shipState, weaponConfig, ownerId, id, currentTick)`:
   - velocity = ship velocity + heading * bulletSpeed
   - heading from orientation: cos(orientation * 2PI), sin(orientation * 2PI)
   - endTick = currentTick + BULLET_ALIVE_TIME
   - bouncesRemaining = 0

2. `createBomb(shipState, weaponConfig, ownerId, id, currentTick)`:
   - Same velocity logic but with bombSpeed
   - endTick = currentTick + BOMB_ALIVE_TIME
   - bouncesRemaining = weaponConfig.bombBounceCount
   - Returns { projectile, recoilVx, recoilVy } so caller can apply recoil to ship

3. `updateProjectile(proj, map, dt)`:
   - Move X axis: proj.x += proj.vx * dt
   - Check wall collision with isCollidingWithWalls(proj.x, proj.y, 0.1, map)
   - If colliding: restore x, reverse vx
     - If bullet: return 'wall_explode'
     - If bomb: decrement bouncesRemaining, if <= 0 return 'wall_explode', else 'active'
   - Same for Y axis
   - Return 'active'

4. `checkProjectileHit(proj, playerX, playerY, playerRadius, playerId)`:
   - If proj.ownerId === playerId, return false (no self-hit)
   - AABB overlap: |proj.x - playerX| < (0.1 + playerRadius) AND |proj.y - playerY| < (0.1 + playerRadius)
   - Return true if overlapping

5. `calculateBombDamage(distance, bombLevel)`:
   - explodeRadius = BOMB_EXPLODE_RADIUS * (1 + bombLevel)
   - If distance >= explodeRadius, return 0
   - damage = (explodeRadius - distance) * (BOMB_DAMAGE_LEVEL / explodeRadius)
   - Return Math.floor(damage)

After all tests pass, update `packages/shared/src/index.ts` to export all weapon functions.
  </implementation>
