2. Create `packages/server/src/game-server.ts`:
   - `GameServer` class:
     - Constructor takes `{ map: TileMap, port: number }`
     - Uses `process.hrtime.bigint()` for high-resolution timer
     - `private tickCount = 0`, `private accumulator = 0n` (BigInt nanoseconds)
     - `private playerManager: PlayerManager`
     - `private weaponManager: WeaponManager`
     - `private clients: Map<string, WebSocket>` (playerId -> ws connection)
     - `private inputQueues: Map<string, Array<{seq, data: ShipInput, fire, fireBomb}>>`
   - `start()`: Create `WebSocketServer({ port })`. Set up `wss.on('connection', ...)`. Start game loop with `setInterval(() => this.update(), 1)`.
   - `update()`: Accumulator pattern with NS_PER_TICK = BigInt(Math.floor(1e9 / TICK_RATE)). While accumulator >= NS_PER_TICK, call `tick()`.
   - `tick()`:
     1. For each player with queued inputs: shift input, call `updateShipPhysics(player.ship, input.data, SHIP_CONFIGS[player.shipType], TICK_DT)` then `applyWallCollision(...)`. Set player.lastProcessedSeq = input.seq. If input.fire, weaponManager.fireBullet. If input.fireBomb, weaponManager.fireBomb.
     2. For alive players with empty input queue: still run physics with neutral input ({left:false,right:false,thrust:false,afterburner:false}) — maintains speed clamp.
     3. `weaponManager.update(TICK_DT, tickCount, map, playerManager)`
     4. Check for deaths: if applyDamage returned true, call killPlayer, broadcast DEATH event.
     5. Check for respawns: for dead players where tick >= respawnTick, call respawnPlayer, broadcast SPAWN event.
     6. Every SNAPSHOT_RATE ticks (5): build GameSnapshot, send to all clients as SNAPSHOT message.
     7. Increment tickCount.
   - `onConnection(ws)`: Listen for messages. On JOIN: create player via playerManager, send WELCOME, broadcast PLAYER_JOIN. On INPUT: push to inputQueue. On SHIP_SELECT: update player shipType. On PING: send PONG.
   - `onDisconnect(playerId)`: broadcast PLAYER_LEAVE, schedule removal after RECONNECT_TIMEOUT.
   - `broadcastSnapshot()`: Build GameSnapshot from playerManager.getAllPlayers() and weaponManager.getProjectiles(). JSON.stringify and send to all clients.
   - `broadcast(msg)`: Send to all connected clients.
   - `send(playerId, msg)`: Send to specific client.
