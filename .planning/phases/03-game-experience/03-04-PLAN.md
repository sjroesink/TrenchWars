    2. Create packages/client/src/audio/sound-manager.ts -- SoundManager class:
       - Import `{ Howl }` from `'howler'`
       - Class with `private sfx: Howl | null = null;`
       - `private initialized = false;`
       - `private thrustId: number | null = null;` -- track looping thrust sound
       - `init(): void` -- creates the Howl instance. Called on first user interaction (ship-select click/key):
         ```typescript
         this.sfx = new Howl({
           src: ['/audio/sfx.webm', '/audio/sfx.mp3'],
           sprite: {
             thrust:     [0, 200, true],     // looping
             bulletFire: [200, 150],
             bombFire:   [350, 300],
             explosion:  [650, 500],
             bounce:     [1150, 100],
             death:      [1250, 400],
           },
           volume: 0.5,
         });
         this.initialized = true;
         ```
       - `play(name: string, pan?: number): void`:
         - If not initialized or sfx is null, return silently
         - `const id = this.sfx.play(name);`
         - If pan is defined: `this.sfx.stereo(pan, id);`
       - `startThrust(): void` -- if not already playing: `this.thrustId = this.sfx?.play('thrust') ?? null;`
       - `stopThrust(): void` -- if playing: `this.sfx?.stop(this.thrustId!); this.thrustId = null;`
       - `isInitialized(): boolean { return this.initialized; }`

       NOTE: The actual audio sprite files (/audio/sfx.webm, /audio/sfx.mp3) will need to be created separately as placeholder files. For now, create a silent/minimal placeholder. The SoundManager should gracefully handle missing audio files (Howler.js does this -- it just logs a warning).

    3. Create placeholder audio directory and a note file:
       ```bash
       mkdir -p packages/client/public/audio
       ```
       Create a placeholder note at `packages/client/public/audio/README.txt`:
       "Audio sprite files (sfx.webm, sfx.mp3) should be generated from individual sound effects using the audiosprite CLI tool. See 03-RESEARCH.md for details."

    4. Create packages/client/src/visual-effects.ts -- VisualEffects class:
       - Import `{ Graphics, Container }` from `'pixi.js'`
       - Manages engine exhaust particles and enhanced explosions
       - Constructor takes `parent: Container`
       - Creates internal Graphics object added to parent
       - Internal arrays for particles and explosions

       Engine exhaust:
       - `private exhaustParticles: { x: number; y: number; vx: number; vy: number; life: number; maxLife: number; size: number }[]`
       - `spawnExhaust(shipX: number, shipY: number, orientation: number): void`:
         - Spawn 1 particle behind ship (opposite of facing direction)
         - Position: ship position
         - Velocity: opposite of ship orientation with slight random spread (+-0.2 radians), speed ~2-4 tiles/s
         - Size: random 2-4px radius
         - Lifetime: 200-400ms (store as frames at 60fps: 12-24 frames)

       Enhanced explosions (replaces basic WeaponRenderer explosions):
       - `private explosions: { x: number; y: number; frame: number; maxFrames: number; large: boolean }[]`
       - `addExplosion(x: number, y: number, large: boolean): void`:
         - Small (bullet kill): maxFrames = 18 (300ms at 60fps), start radius 4px, end 16px
         - Large (bomb kill): maxFrames = 30 (500ms at 60fps), start radius 8px, end 32px
       - `render(camera: Camera): void`:
         - Clear graphics
         - For each exhaust particle:
           - Progress = life / maxLife
           - Color: lerp from `0x00ff88` to `0x004422` based on progress
           - Alpha: 1.0 - progress (fade out)
           - Draw circle at camera.worldToScreen(x, y) with size
           - Update: x += vx * dt, y += vy * dt, life++
           - Remove if life >= maxLife
         - For each explosion:
           - Progress = frame / maxFrames
           - Inner circle (white 0xffffff) at small radius, alpha = (1-progress)
           - Outer circle (0xffaa33) at medium radius, alpha = (1-progress) * 0.7
           - Edge circle (0xff6600) at full radius, alpha = (1-progress) * 0.4
           - Draw at camera.worldToScreen position
           - Increment frame, remove if done
       - `destroy(): void`
