**Implementation -- `packages/shared/src/physics.ts`:**

Functions to export:

1. `applyRotation(state: ShipState, input: ShipInput, config: ShipConfig, dt: number): void`
   - Left: `state.orientation -= config.rotation * dt`
   - Right: `state.orientation += config.rotation * dt`
   - Wrap: `if < 0, += 1; if >= 1, -= 1`

2. `applyThrust(state: ShipState, input: ShipInput, config: ShipConfig, dt: number): void`
   - If not thrusting, return (do NOT apply drag -- this is critical)
   - Choose thrust value: `input.afterburner && state.energy > 0 ? config.maxThrust : config.thrust`
   - `angle = state.orientation * Math.PI * 2`
   - `state.vx += Math.cos(angle) * thrustValue * dt`
   - `state.vy += Math.sin(angle) * thrustValue * dt`

3. `clampSpeed(state: ShipState, input: ShipInput, config: ShipConfig): void`
   - Choose speed limit: `input.afterburner && state.energy > 0 ? config.maxSpeed : config.speed`
   - `currentSpeed = Math.sqrt(vx*vx + vy*vy)`
   - If `currentSpeed > speedLimit`: scale both vx, vy by `speedLimit / currentSpeed`
   - NEVER reduce speed below current if under limit (this is Truncate, not drag)

4. `updateEnergy(state: ShipState, input: ShipInput, config: ShipConfig, dt: number): void`
   - If afterburner active AND energy > 0: `state.energy -= config.afterburnerCost * dt`
   - Always recharge: `state.energy += config.recharge * dt`
   - Clamp: `state.energy = Math.max(0, Math.min(config.maxEnergy, state.energy))`

5. `updateShipPhysics(state: ShipState, input: ShipInput, config: ShipConfig, dt: number): void`
   - Calls in order: applyThrust, applyRotation, clampSpeed, updateEnergy
   - (Position update and collision handled separately in collision.ts)

CRITICAL: Do NOT apply any velocity decay/drag. When no input is pressed, velocity must remain EXACTLY unchanged. This is the defining characteristic of SubSpace physics.

**collision.ts -- Functions:**

1. `isWallAt(map: TileMap, tileX: number, tileY: number): boolean`
   - Out of bounds (x < 0, x >= width, y < 0, y >= height) returns true
   - `map.tiles[tileY * map.width + tileX] !== 0`

2. `isCollidingWithWalls(x: number, y: number, radius: number, map: TileMap): boolean`
   - Check all tiles in range: x +/- radius, y +/- radius
   - For each tile in range: if isWallAt AND box-box intersection (ship AABB vs tile AABB), return true
   - Ship AABB: [x-radius, y-radius] to [x+radius, y+radius]
   - Tile AABB: [tx, ty] to [tx+1, ty+1]

3. `simulateAxis(state: ShipState, dt: number, axis: 'x' | 'y', map: TileMap, radius: number, bounceFactor: number): boolean`
   - Save previous position on this axis
   - Move: `state[axis] += state[velKey] * dt`
   - If colliding: restore position, `state[velKey] *= -bounceFactor`
   - Return true if bounced

4. `applyWallCollision(state: ShipState, dt: number, map: TileMap, radius: number, bounceFactor: number): void`
   - Call simulateAxis for 'x', then for 'y' (axis-separated, matches SubSpace order)
